## 🛠️ Technical Implementation

- **4 Normalized Tables**: Customers, Products, Orders, Order_Items
- **Relationships**:
  - Customers → Orders (1-to-Many)
  - Orders → Order_Items (1-to-Many) 
  - Products → Order_Items (1-to-Many)
